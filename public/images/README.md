# Brand images

Drop the generated photos/logo here with these exact filenames. They're
referenced as `/images/<name>` from the app (Vite serves `public/` at the site
root).

| Filename | Which generated image | Used on | Suggested size |
|---|---|---|---|
| `logo.png` | The "השף שלי" logo — **transparent PNG** | Footer brand | ~600×600, transparent background |
| `hero.png` | Chef's hands plating, close-up (landscape) | Homepage hero | ~1600×1067 (3:2) |
| `og.png` | Candlelit dinner table at dusk (landscape) | Social share card **and** homepage closing band | 1200×630 (1.91:1) |
| `pomegranate.png` | Split pomegranate, seeds spilling (vertical) | Homepage "why" card | ~900×1100 |
| `olive.png` | Olive branch + thyme on linen (landscape) | "For chefs" page banner | ~1600×700 (16:7) |
| `chef.png` | Chef plating in a rustic kitchen (landscape) | "Chef join" page banner | ~1600×700 (16:7) |
| `saffron.png` | Saffron + spices in a ceramic bowl (square) | Spare accent (not placed yet) | ~1000×1000 |
| `texture.png` | Close-up linen weave (landscape) | Spare (subtle section background, not placed yet) | ~1600×900 |

Notes:
- **`logo.png` must be a transparent PNG** so it sits cleanly on the linen footer
  (use the version without the white/cream background box).
- Photos: **JPG or WebP** are both fine; keep each file roughly < 300 KB
  (export quality ~80) so pages stay fast.
- `og.png` should be close to **1200×630** for clean link previews; the same
  image also backs the homepage closing band.
- Until a file is added, that one spot shows blank — everything else renders.
